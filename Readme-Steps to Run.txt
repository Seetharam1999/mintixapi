--------------------------WEGOT API--------------------------------
-------------------------------------------------------------------

Steps to start the server.
1. Unzip the folder.
2. In command prompt, move to the unzipped folder.
3. Run the following commands.
	npm install (To install the npm packages mentioned in the package.json)
	set NODE_Env=development (setting the environment to development)
	npm start
4. Try reaching localhost:3000 in the browser to get the name and version (localhost:3000/sample for sample data)
